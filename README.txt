There were issues with virtual database tracking therefore there are existing problems in running simulation process after setting up team.
Whole project was made using VSCode because of the environment accesible at the time. 
To run Project setting up angular.js in miniclipProj is needed. 
While using terminal head to miniclipProj directory.
Use npm install and there after npm run start.
To start the backend app go to miniclipBackendApp directory in the terminal and use dotnet run command. 
If there are problems use dotnet clean to clean solution and dotnet build to build it again. Then you can run it with dotnet run.
